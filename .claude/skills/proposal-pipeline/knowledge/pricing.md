# Pricing Guidelines

Rate guidelines for proposals. Never quote below minimums.

## Minimums

- **AIIn60 Starter:** $2,000
- **Polymaxr Strategy:** $3,000
- **Hourly (if forced):** $75/hour

## Hourly Rates

| Complexity | Rate Range | Examples |
|------------|------------|----------|
| Simple | $50 - $75/hr | Basic bot setup, Discord/Telegram integrations |
| Standard | $75 - $100/hr | Custom trading bot, API integrations, dashboards |
| Complex | $100 - $125/hr | Multi-strategy systems, AI agents, prediction market bots |
| Expert | $125 - $150/hr | Architecture design, on-chain systems, real-time risk engines |

**Default rate**: $85/hr (covers most standard projects)

## Fixed Price Guidelines

| Custom integration | $1,500 - $3,500 | API work, webhooks, moderate complexity |
| Full trading system | $5,000 - $10,000 | Multi-strategy, dashboard, risk management |

- Multi-market MM with dashboard: $6,000 - $12,000
- Arbitrage/scalping system: $4,000 - $8,000
- Full platform: $15,000+

## AI Agent Project Pricing

- Basic chatbot with tools: $1,500 - $3,000
- Multi-agent system: $4,000 - $8,000
- Custom skill development: $500 - $1,500 per skill
- Agent deployment + hosting setup: $500 - $1,000

## Adjustments

### Discounts
- **Long-term retainer** (6+ months): -10-15%
- **Portfolio-worthy project**: up to -20%
- **Referral source**: -10%
- **Quick scope, easy client**: small discount for low-friction work

### Premiums
- **Rush timeline** (< 2 weeks): +50%
- **Weekend/holiday work**: +50%
- **Scope uncertainty**: +20% buffer
- **Complex integrations**: price the complexity, not the time

## Negotiation Notes

- Always ask about budget range first
- Provide value-based justification (ROI for trading systems)
- Offer milestone-based payments for projects > $2,000
- Include 1-2 revision rounds in fixed-price quotes
- For hourly: propose weekly caps for budget certainty
- Escrow preferred for new clients

## Budget Translation

When the job posting says:
- "Budget: $X" -> They may have more. Quote value, not floor.
- "Fixed price" -> Quote with milestones, 50% upfront.
- "Hourly" -> Estimate hours, propose fixed if possible.
- No budget mentioned -> Ask in proposal, don't guess.
